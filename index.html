Tema 2: Elementos de la Estructuración de una Página Web

1. Contenedor (wrapper)

El contenedor o "wrapper" es un elemento que agrupa el contenido de la página. Suele tener un ancho máximo para centrar el contenido y mantener un diseño uniforme.

2. Cabecera (header)

La cabecera de una página web se usa para incluir el logo, el título y cualquier información de introducción. Suele estar en la parte superior de la página.

3. Contenido (content)

La sección de contenido es el área principal de la página donde se presenta la información más relevante para los usuarios. Puede incluir textos, imágenes, videos y otros elementos.

4. Menú (menu)

El menú es un componente de navegación que permite a los usuarios moverse entre las diferentes secciones o páginas de un sitio web. Suele estar ubicado en la cabecera o en un lateral.

5. Pie (footer)

El pie de página o "footer" contiene información adicional como derechos de autor, enlaces de contacto y enlaces a políticas de privacidad.

6. Lateral (sidebar)

El lateral o "sidebar" es una sección adicional que puede contener enlaces, widgets, información complementaria o publicidad. Suele estar en un lateral de la página.

7. Etiqueta div

La etiqueta <div> es un contenedor genérico utilizado para agrupar otros elementos y aplicar estilos. No tiene un significado semántico por sí misma.

Tema 3: Estructuración Semántica de una Página Web

1. <header>

La etiqueta <header> define el encabezado de una página o de una sección. Generalmente incluye el logotipo, el título y la navegación principal.

2. <nav>

La etiqueta <nav> define un conjunto de enlaces de navegación. Se utiliza para agrupar los menús de navegación.

3. <main>

La etiqueta <main> representa el contenido principal de un documento. Debe ser única en la página y no debe contener elementos de navegación o de pie de página.

4. <article>

La etiqueta <article> se utiliza para representar un contenido independiente y autónomo que puede ser distribuido o reutilizado, como una entrada de blog o una noticia.

5. <section>

La etiqueta <section> define secciones en un documento, como capítulos o agrupaciones temáticas. Cada sección puede tener un título.

6. <aside>

La etiqueta <aside> representa contenido adicional o relacionado que puede ser considerado como una sección secundaria o complementaria del contenido principal.

7. <footer>

La etiqueta <footer> define el pie de página de un documento o sección. Suele contener información como derechos de autor, enlaces de contacto, y otros datos relevantes.
